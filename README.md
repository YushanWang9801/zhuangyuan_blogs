# 唐宋状元殿试卷解读合集
## 项目简介
本项目收录了唐宋时期7位状元的经典殿试卷（策问/赋/诗），以统一HTML格式呈现「原文+注释+深度解读+播客字幕」（黄裳卷无播客），系统展现科举制度下士人的政治智慧、文学素养与时代思想，为科举文化研究、古典文学学习提供权威且易读的资料合集。

## 项目目标
1. 整理唐宋科举典范文献，还原状元殿试卷的原始风貌与历史语境；
2. 通过原文注释、逻辑解读，降低古典文献阅读门槛；
3. 结合播客字幕（部分），以听觉+视觉双重维度传递经典价值；
4. 呈现唐宋科举从「诗赋取士」到「经世致用」的文风演变轨迹。

## 内容结构
每篇HTML文件统一包含3大核心模块（黄裳卷无播客字幕模块）：
| 模块         | 核心内容                                                                 |
|--------------|--------------------------------------------------------------------------|
| 原文+注释    | 作者简介、殿试题目、状元答卷原文、生僻字词/典故注释，确保文本准确性与可读性 |
| 深度解读     | 题意解析（帝王命题意图）、答卷解读（逻辑结构+思想内涵）、整体价值（政治/文学/历史） |
| 播客字幕     | 双人对话形式，通俗解读创作背景、核心观点、文化价值，适配音频同步阅读       |
